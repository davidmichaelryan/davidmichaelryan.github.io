Other work

I've done some other cool projects while at Northwestern - here are some I'm particularly proud of:

  • Mayfest

    In spring 2015, I converted a simple html website for Mayfest's Dillo Day into a full-fledged, CMS-backed Node.js website using Keystone.js. It uses modern developer tools including Sass, Coffeescript, Mustache, and Grunt to create a modern, future-forward site for the festival.

    Earlier, I was the sole developer for the first ever Dillo Day iOS app. I managed to hack it together in Spring 2014 using Appgyver's Steroids IDE.

  • A&O

    I work on the website for A&O Productions, a student group on campus that brings artists and speakers to campus. One of my main tasks is designing cool splash pages for their announcements. My favorite is from this past spring, when NUDM and A&O teamed up for a Benefit Concert that featured Magic Man and Cheers Elephant. I've saved the splash page here.

  • Random Pocket
